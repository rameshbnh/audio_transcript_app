History Button Final Animation

This demo shows the final history button animation working exactly as you requested:

  1. Initially, the "Audio Gateway UI" header, upload card, and history button are all visible
  2. When "Show History" is clicked:
    • Both the upload card and history button slide UP together
    • They completely disappear from view (move -100vh)
    • The "Audio Gateway UI" header remains visible at the top
    • The "Hide History" button appears right below the header
    • The history panel expands below the button
  3. When "Hide History" is clicked:
    • The history panel collapses
    • The upload card and history button slide back DOWN
    • Everything returns to its original position

How It Works

/* CSS for animations */
.content-wrapper {
  position: relative;
  height: 400px;
  overflow: hidden; /* Clips content that goes outside */
}

.animated-section {
  transition: transform 0.5s ease-in-out;
  margin-top: 20px;
}

.animated-section.slide-up {
  transform: translateY(-100vh); /* Completely off-screen */
}

.history-section {
  transition: max-height 0.5s ease-in-out;
  max-height: 0;
  overflow: hidden;
}

.history-section.expanded {
  max-height: 1000px;
  overflow: visible;
}
