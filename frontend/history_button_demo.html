History Button Animation

This demo shows the history button animation working exactly as you requested:

  1. History button starts below the upload card
  2. When "Show History" is clicked:
    • Upload card slides up
    • History button moves to the top
    • History panel expands below the button
  3. When "Hide History" is clicked:
    • History panel collapses
    • Upload card slides back down
    • History button moves back to original position

How It Works

/* CSS for animations */
.upload-section {
  transition: transform 0.5s ease-in-out;
}

.upload-section.slide-up {
  transform: translateY(-100%);
}

.history-toggle-btn {
  transition: all 0.5s ease-in-out;
}

.history-toggle-btn.move-to-top {
  position: absolute;
  top: 0;
  left: 28px;
  width: 344px;
  margin-top: 0;
}

.history-section {
  transition: max-height 0.5s ease-in-out;
  max-height: 0;
  overflow: hidden;
}

.history-section.slide-down {
  max-height: 1000px;
  overflow: visible;
}
